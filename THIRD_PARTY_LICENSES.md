# Лицензии сторонних компонентов

Качалка запускает и/или собирает внутрь Docker-образа ряд сторонних
инструментов. Ниже — их лицензии. Каждый компонент запускается как отдельный
процесс (через `exec()`, CLI или локальный HTTP), их исходный код не
изменяется и не встраивается в код этого репозитория — то есть их лицензии
не распространяются на собственный код проекта (см. LICENSE), за исключением
случая, отдельно оговорённого ниже.

## yt-dlp
- Репозиторий: https://github.com/yt-dlp/yt-dlp
- Лицензия: **Unlicense** (эквивалент public domain)
- Использование: устанавливается через pip, вызывается как CLI-процесс

## ffmpeg
- Репозиторий: https://ffmpeg.org
- Лицензия: **LGPL / GPL** (зависит от флагов сборки конкретного пакета)
- Использование: системный пакет Debian, вызывается как CLI-процесс для
  муксинга/конвертации; не модифицируется и не линкуется

## vot-cli
- Репозиторий: https://github.com/FOD-Grup/vot-cli
- Лицензия: **MIT**
- Использование: устанавливается глобально через npm, вызывается как
  CLI-процесс для получения переведённой озвучки

## bgutil-ytdlp-pot-provider
- Репозиторий: https://github.com/Brainicism/bgutil-ytdlp-pot-provider
- Лицензия: **GPL-3.0-or-later**
- Использование: исходный код клонируется без изменений и компилируется на
  этапе сборки образа (`npx tsc`); собранный сервер запускается как отдельный
  локальный процесс на 127.0.0.1:4416 и общается с yt-dlp по HTTP.
  Программа остаётся физически отдельной (собственный процесс, порт,
  протокол) — код этого репозитория с ней не линкуется и не является
  производной работой. Исходный код компонента доступен в неизменном виде
  по ссылке выше.

## nginx
- Сайт: https://nginx.org
- Лицензия: **BSD-2-Clause**
- Использование: системный пакет Debian, веб-сервер и reverse proxy

## PHP
- Сайт: https://www.php.net
- Лицензия: **PHP License 3.01**
- Использование: системный пакет Debian, среда выполнения бэкенда

## Node.js
- Сайт: https://nodejs.org
- Лицензия: **MIT** (с компонентами под другими permissive-лицензиями)
- Использование: устанавливается из NodeSource, требуется для vot-cli и
  сборки bgutil-ytdlp-pot-provider

---

Список актуален на момент составления и может не отражать более поздние
изменения версий зависимостей в Dockerfile. При сомнениях сверяйтесь с
LICENSE-файлом соответствующего пакета в его официальном репозитории.
