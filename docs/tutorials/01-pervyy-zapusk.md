# Первый запуск

Этот туториал проведёт тебя за руку от пустой папки до первого скачанного видео. Ничего настраивать заранее не нужно - только Docker.

Если Docker ещё не установлен, поставь его с [официального сайта](https://docs.docker.com/get-docker/) и возвращайся сюда.

## Шаг 1. Запусти готовый образ

Собирать ничего не нужно - готовый образ уже лежит на [Docker Hub](https://hub.docker.com/r/mikedigriz/yt):

```bash
docker run -it -p 8000:80 mikedigriz/yt
```

Команда скачает образ (если его ещё нет на компьютере), запустит контейнер и пробросит порт 8000 наружу.

Готово: сайт уже работает.

Если позже понадобится прокси, его можно передать прямо переменной окружения, без файла `.env`:

```bash
docker run -it -p 8000:80 -e SOCKS5_URL=socks5://user:pass@host:port mikedigriz/yt
```

## Шаг 2. Открой сайт в браузере

Перейди на [http://localhost:8000](http://localhost:8000). Ты увидишь поле для ссылки и кнопку **Скачать**.

## Шаг 3. Скачай первое видео

1. Найди любое видео на YouTube, VK или другом сайте, который поддерживает yt-dlp, и скопируй на него ссылку.
2. Вставь ссылку в поле на главной странице.
3. Нажми **Скачать**.
4. Перейди на вкладку **Загрузки** - там видно прогресс в реальном времени.
5. Когда загрузка закончится, перейди на вкладку **Видео** и забери файл по ссылке или QR-коду.

Вот и всё - у тебя есть рабочая Качалка на своём сервере.
