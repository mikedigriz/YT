# Первый запуск

Этот туториал проведёт тебя за руку от пустой папки до первого скачанного видео. Ничего настраивать заранее не нужно - только Docker.

Если Docker ещё не установлен, поставь его с [официального сайта](https://docs.docker.com/get-docker/) и возвращайся сюда.

## Шаг 1. Скачай проект

```bash
git clone https://github.com/mikedigriz/YT.git
cd YT
```

Команда скопирует репозиторий Качалки в папку `YT` и перейдёт в неё.

## Шаг 2. Подготовь настройки

```bash
cp .env.example .env
```

Файл `.env` хранит настройки, которые не должны попадать в git - например, пароль от прокси. Для первого запуска ничего в нём менять не нужно: пустой `SOCKS5_URL` просто означает "прокси не используем".

## Шаг 3. Собери и запусти контейнер

```bash
docker build -t yt .
docker run -it -p 8000:80 yt
```

Первая команда собирает образ - ставит внутрь PHP, nginx, Python, Node.js, ffmpeg и сам yt-dlp. Это может занять несколько минут, потерпи. Вторая команда запускает контейнер и пробрасывает порт 8000 наружу.

Готово: сайт уже работает.

## Шаг 4. Открой сайт в браузере

Перейди на [http://localhost:8000](http://localhost:8000). Ты увидишь поле для ссылки и кнопку **Скачать**.

## Шаг 5. Скачай первое видео

1. Найди любое видео на YouTube, VK или другом сайте, который поддерживает yt-dlp, и скопируй на него ссылку.
2. Вставь ссылку в поле на главной странице.
3. Нажми **Скачать**.
4. Перейди на вкладку **Загрузки** - там видно прогресс в реальном времени.
5. Когда загрузка закончится, перейди на вкладку **Видео** и забери файл по ссылке или QR-коду.

Вот и всё - у тебя есть рабочая Качалка на своём сервере.
