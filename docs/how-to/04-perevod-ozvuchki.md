# Включить перевод озвучки

Качалка умеет переводить закадровую озвучку иностранного видео на русский язык - тем же движком, что использует Яндекс.Браузер (VOT, Video Translation). Перевод работает только для видео на публичных сайтах, которые знает Яндекс (YouTube, VK, Vimeo и похожие) и только для **иностранной** речи - русскоязычное видео переводить нечего, вернётся оригинал.

## Как включить

1. Вставь ссылку на иностранное видео в поле на главной странице.
2. Зажми и удерживай кнопку **Скачать** дольше полусекунды - откроется меню с выбором качества.
3. В этом меню выбери пункт **Перевод**.
4. Отпусти кнопку над этим пунктом - загрузка запустится с переводом.

Работает только для видео, для аудио пункт смысла не имеет.

## Что происходит после запуска

Пока идёт перевод, вкладка "Загрузки" покажет не просто "В процессе", а конкретную фазу:

1. **Перевожу озвучку через Яндекс** - в это время yt-dlp качает оригинал, а сервис Яндекса параллельно готовит русскую дорожку.
2. **Вклеиваю русскую дорожку в видео** - ffmpeg склеивает перевод с видео, русская дорожка становится основной, оригинал остаётся второй.

Готовый файл получит суффикс `_ru.mp4`.

Если перевод не удался (Яндекс не знает сайт, видео уже русское, сервис недоступен) - ты всё равно получишь обычное скачанное видео без ошибки, просто без перевода.

## Если перевод не работает

Симптом - любой перевод даёт ошибку вида "unknown service" или "перевод не найден", хотя само видео качается нормально. Причина почти всегда одна: Яндекс сменил внутренний ключ доступа к сервису перевода, и используемый инструмент (`vot-cli`) отстал от изменений.

Проверить вручную (подставь реальный id видео):

```bash
docker exec yt vot-cli --reslang=ru 'https://www.youtube.com/watch?v=<id>'
```

Если команда тоже не находит перевод - дело не в Качалке, а в самом сервисе.

Починить: пересобрать образ, чтобы подтянуть свежую версию инструмента:

```bash
docker build --no-cache -t yt . && docker-compose up -d
```

Либо на горячую, без пересборки образа:

```bash
docker exec -u root yt npm install -g vot-cli
```

Как устроен пайплайн перевода изнутри - в [Как устроены перевод и обход бот-чека YouTube](../explanation/06-youtube-i-perevod-kak-eto-rabotaet.md).
