# Документация проекта "Качалка"

Качалка - это веб-интерфейс для скачивания видео и аудио с разных сайтов. Под капотом работает yt-dlp, а Качалка добавляет к нему удобную страницу, очередь загрузок, систему уведомлений и кучу мелких удобств.

Эта документация разбирает проект изнутри: как он устроен, что делает каждая важная функция и почему она написана именно так. Если ты хочешь просто разобраться, как пользоваться сайтом, - читай раздел "Как это работает" в разделах ниже. Если хочешь залезть в код - там же найдёшь разбор по функциям.

## Что умеет Качалка

- Скачивает видео и аудио по ссылке, в том числе несколько ссылок сразу через `||`
- Конвертирует видео в аудио (mp3, wav, aac, flac)
- Умеет выбирать качество видео: обычное (до потолка из настроек), и через долгое нажатие на кнопку "Скачать" - явно 4K, 2K или 1080p
- Держит очередь загрузок, если сервер уже занят
- Понимает 50+ типов ошибок yt-dlp и показывает их по-русски и с юмором
- Сама удаляет старые файлы через 2 часа
- Поддерживает прокси SOCKS5 для сайтов, которые блокируют российские IP
- Показывает иконку сайта прямо в поле ввода ссылки, пока ты её печатаешь
- Имеет пасхалку с лазерным Снежиком (см. [Пасхалки](07-easter-eggs.md))

## Структура проекта

```
YT/
├── app/
│   ├── start.sh                 точка входа контейнера
│   ├── logger.sh                генерирует Python-плагин логирования yt-dlp
│   ├── etc/
│   │   ├── nginx/                конфиг веб-сервера
│   │   ├── yt-dlp.conf            общий конфиг yt-dlp (лимиты, фильтр 18+)
│   │   └── Scripts/               скрипты очистки старых файлов
│   ├── patches/                  патч для yt-dlp (имена файлов)
│   └── var/www/YT/               сама веб-папка приложения
│       ├── index.php              точка входа, обработка запросов
│       ├── error_pages.php        страница ошибки CSRF
│       ├── class/
│       │   ├── Downloader.php     запуск и управление загрузками
│       │   ├── FileHandler.php    работа с файлами на диске
│       │   └── Session.php        обёртка над сессией PHP
│       ├── config/
│       │   ├── config.php          настройки сайта
│       │   └── favicon_domains.json список сайтов для подсказки иконок
│       ├── views/                  HTML-шаблоны страницы
│       ├── js/                     фронтенд-логика
│       ├── css/                    стили и темы
│       ├── download/               сюда сохраняются скачанные файлы
│       └── tmp/                    логи и очередь загрузок
├── Dockerfile
├── docker-compose.yml
└── .env.example
```
